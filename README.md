# GESTEC

Sistema de gestão escolar para organização da rotina da ETEC: login da gestão, solicitação/aprovação de professores, mural de avisos, status de salas, substituições, horários, reservas e perfil do professor.

## Stack

- Next.js 16 App Router
- React 19
- Tailwind CSS 4
- Banco de dados relacional
- Vercel

## Rodar localmente

```bash
npm install
npm run dev
```

Acesse `http://localhost:3000`.

## Configuração segura

As credenciais, chaves de banco de dados e dados administrativos devem ficar apenas nos ambientes seguros de desenvolvimento e produção. Não publique essas informações no repositório.

## Verificação

```bash
npm run lint
npm run build
```
